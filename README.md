# 🎮 Tic Tac Toe AI

An interactive Tic Tac Toe game built with **Python** and **Streamlit**, featuring an unbeatable AI opponent powered by the **Minimax Algorithm**.

## 📌 Project Overview

This project allows users to play Tic Tac Toe against an AI that makes optimal decisions using the Minimax algorithm. The application provides a clean and responsive interface with a custom Lemon Chiffon and Ultra Violet color theme.

## ✨ Features

* Human vs AI gameplay
* Unbeatable AI using Minimax Algorithm
* Interactive and responsive user interface
* Real-time game status updates
* Restart game functionality
* Custom aesthetic theme
* Built entirely with Python and Streamlit

## 🛠️ Technologies Used

* Python
* Streamlit
* Minimax Algorithm
* HTML/CSS Styling (via Streamlit)

## 🧠 AI Implementation

The AI uses the **Minimax Algorithm**, a decision-making algorithm commonly used in two-player games.

### How It Works

* The algorithm evaluates all possible future game states.
* It assumes the AI plays optimally.
* It selects the move with the highest possible score.
* The AI can never lose if implemented correctly.

### Scoring Logic

| Result     | Score |
| ---------- | ----- |
| AI Wins    | +10   |
| Human Wins | -10   |
| Draw       | 0     |

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/tic-tac-toe-ai.git
cd tic-tac-toe-ai
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
```

### 3. Activate the Virtual Environment

Windows:

```bash
.\venv\Scripts\Activate.ps1
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the Application

```bash
streamlit run app.py
```
